## List All Skill Categories

<Note>
  **Authentication Required**: This endpoint requires a valid JWT token.
</Note>

```http
GET /api/me/skill-categories
```

Retrieve all skill categories with their associated skills for the authenticated user.

### Response

<ResponseField name="status" type="string">
  Response status ("success" or "error")
</ResponseField>

<ResponseField name="message" type="string">
  Response message
</ResponseField>

<ResponseField name="data" type="array">
  Array of skill categories
  
  <ResponseField name="id" type="long">
    Skill category ID
  </ResponseField>
  
  <ResponseField name="name" type="string">
    Category name
  </ResponseField>
  
  <ResponseField name="sortOrder" type="integer">
    Display order for sorting
  </ResponseField>
  
  <ResponseField name="skills" type="array">
    Array of skills in this category
    
    <ResponseField name="id" type="long">
      Skill ID
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Skill name
    </ResponseField>
    
    <ResponseField name="globalSkillId" type="long">
      Reference to global skill database
    </ResponseField>
    
    <ResponseField name="level" type="integer">
      Proficiency level (0-100)
    </ResponseField>
    
    <ResponseField name="icon" type="string">
      Icon identifier for the skill
    </ResponseField>
  </ResponseField>
</ResponseField>

<CodeGroup>
```bash cURL
curl -X GET https://api.portfoliohub.com/api/me/skill-categories \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

```json Response
{
  "status": "success",
  "message": "Categorías de skills obtenidas",
  "data": [
    {
      "id": 1,
      "name": "Programming Languages",
      "sortOrder": 1,
      "skills": [
        {
          "id": 10,
          "name": "Java",
          "globalSkillId": 100,
          "level": 90,
          "icon": "java-icon"
        },
        {
          "id": 11,
          "name": "JavaScript",
          "globalSkillId": 101,
          "level": 85,
          "icon": "js-icon"
        }
      ]
    }
  ]
}
```
</CodeGroup>

## Batch Create Skill Categories

  **Authentication Required**: This endpoint requires a valid JWT token.
</Note>

```http

```

Create multiple skill categories at once.

### Request Body

<ParamField body="categories" type="array" required>
  Array of category creation requests
  
  <ParamField body="name" type="string" required>
    Category name (max 80 characters)
  </ParamField>
  
  <ParamField body="sortOrder" type="integer" required>
    Display order for sorting
  </ParamField>
</ParamField>

### Response

<ResponseField name="status" type="string">
  Response status ("success" or "error")
</ResponseField>

<ResponseField name="message" type="string">
  Response message
</ResponseField>

<ResponseField name="data" type="array">
  Array of created skill categories
</ResponseField>

<CodeGroup>
```bash cURL
curl -X POST https://api.portfoliohub.com/api/me/skill-categories/batch \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \

  -d '[
    {
      "name": "Programming Languages",
```
