## List All Projects

<Note>
  **Authentication Required**: This endpoint requires a valid JWT token.
</Note>

```http
GET /api/me/projects
```

Retrieve all project entries for the authenticated user.

### Response

<ResponseField name="status" type="string">
  Response status ("success" or "error")
</ResponseField>

<ResponseField name="message" type="string">
  Response message
</ResponseField>

<ResponseField name="data" type="array">
  Array of project entries
  
  <ResponseField name="id" type="long">
    Project ID
  </ResponseField>
  
  <ResponseField name="title" type="string">
    Project title
  </ResponseField>
  
  <ResponseField name="slug" type="string">
    URL-friendly project slug
  </ResponseField>
  
  <ResponseField name="summary" type="string">
    Brief project summary
  </ResponseField>
  
  <ResponseField name="description" type="string">
    Detailed project description
  </ResponseField>
  
  <ResponseField name="repoUrl" type="string">
    Repository URL (e.g., GitHub)
  </ResponseField>
  
  <ResponseField name="liveUrl" type="string">
    Live demo URL
  </ResponseField>
  
  <ResponseField name="coverImage" type="string">
    Cover image URL
  </ResponseField>
  
  <ResponseField name="startDate" type="string">
    Project start date (ISO 8601 format: YYYY-MM-DD)
  </ResponseField>
  
  <ResponseField name="endDate" type="string">
    Project end date (ISO 8601 format: YYYY-MM-DD)
  </ResponseField>
  
  <ResponseField name="featured" type="boolean">
    Whether this project is featured
  </ResponseField>
  
  <ResponseField name="sortOrder" type="integer">
    Display order for sorting
  </ResponseField>
  
  <ResponseField name="skills" type="array">
    Array of associated skills
    
    <ResponseField name="id" type="long">
      Skill ID
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Skill name
    </ResponseField>
    
    <ResponseField name="globalSkillId" type="long">
      Global skill reference ID
    </ResponseField>
    
    <ResponseField name="level" type="integer">
      Skill proficiency level (0-100)
    </ResponseField>
    
    <ResponseField name="icon" type="string">
      Skill icon identifier
    </ResponseField>
  </ResponseField>
</ResponseField>

<CodeGroup>
```bash cURL
curl -X GET https://api.portfoliohub.com/api/me/projects \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

## Create Project

<Note>
  **Authentication Required**: This endpoint requires a valid JWT token.
</Note>

```http
POST /api/me/projects
```

Create a new project entry.

### Request Body

<ParamField body="title" type="string" required>
  Project title (max 140 characters)
</ParamField>

<ParamField body="summary" type="string" required>
  Brief project summary (max 280 characters)
</ParamField>

<ParamField body="description" type="string">
  Detailed project description (max 65535 characters)
</ParamField>

<ParamField body="repoUrl" type="string">
  Repository URL (max 512 characters)
</ParamField>

<ParamField body="liveUrl" type="string">
  Live demo URL (max 512 characters)
</ParamField>

<ParamField body="coverImage" type="string">
  Cover image URL (max 512 characters)
</ParamField>

<ParamField body="startDate" type="string">
  Project start date in ISO 8601 format (YYYY-MM-DD)
</ParamField>

<ParamField body="endDate" type="string">
  Project end date in ISO 8601 format (YYYY-MM-DD)
</ParamField>

<ParamField body="featured" type="boolean" required>
  Whether this project should be featured
</ParamField>

<ParamField body="sortOrder" type="integer" required>
  Display order for sorting projects
</ParamField>

### Response

<ResponseField name="status" type="string">

</ResponseField>

</ResponseField>

<ResponseField name="data" type="object">

## Associate Skills to Project

<Note>
  **Authentication Required**: This endpoint requires a valid JWT token.
</Note>

```http
POST /api/me/projects/{projectId}/skills
```

Associate multiple skills with a project.

### Path Parameters

<ParamField path="projectId" type="long" required>
  Project ID to associate skills with
</ParamField>

### Request Body

<ParamField body="skillIds" type="array" required>
  Array of skill IDs to associate with the project
</ParamField>

### Response

<ResponseField name="status" type="string">
  Response status ("success" or "error")
</ResponseField>

<ResponseField name="message" type="string">
  Response message
</ResponseField>

<ResponseField name="data" type="object">
  Updated project with associated skills
</ResponseField>

<CodeGroup>
```bash cURL
curl -X POST https://api.portfoliohub.com/api/me/projects/1/skills \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "skillIds": [10, 15, 22]
```
